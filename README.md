# IntermosCraft

This folder is the home for **IntermosCraft** work in this workspace: reference docs, agent briefs, decisions, and other shared info.

## Suggested layout

Create subfolders as needed, for example:

- `docs/` — specifications, architecture notes, runbooks
- `agents/` — prompts, playbooks, and handoff notes for AI agents
- `assets/` — files that belong to the project but are not source code

The Cursor rule **IntermosCraft workspace** (`.cursor/rules/intermoscraft-workspace.mdc`) tells agents to treat this workspace as IntermosCraft and to prefer this directory for durable context.
